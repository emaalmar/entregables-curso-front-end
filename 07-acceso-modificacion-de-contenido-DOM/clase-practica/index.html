Acceder a clases CSS.... Utiliza el objeto .classList para acceder a las clases de un elemento HTML. Consulta cuántas clases tiene, conviértelas en un array y accede a una clase específica.

Añadir y eliminar clases CSS Añade nuevas clases CSS a un elemento utilizando .classList.add() y elimínalas con .classList.remove(). Observa cómo cambia la lista de clases.

Usa .classList.contains() para verificar si un elemento tiene una clase específica.

Alternar clases CSS Utiliza .classList.toggle() para alternar una clase en un elemento. Añade la clase si no existe y elimínala si ya está presente.

Reemplazar una clase CSS Utiliza .classList.replace() para reemplazar una clase existente por otra nueva.
